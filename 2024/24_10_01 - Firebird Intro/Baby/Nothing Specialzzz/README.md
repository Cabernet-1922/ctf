Check [Office Open XML](https://en.wikipedia.org/wiki/Office_Open_XML), you will know that office file are using `.xml` format. So, we may want to unzip the file to check the file inside.\
You will get a flag file by using command `unzip flag.docx > flag`, you can get a flag file in the same directory. Flag is in the second line of the flag file.
